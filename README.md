## Prerequisites
Node version:
```
~$: nvm install 8.12.0
~$: node -v
v8.12.0
```
npm version:
```
~$: npm -v
6.4.0
```
### Installation
sudo ./install-node_8.12.0.sh
```
git clone https://github.com/SmugglerSMR/CAB432-assgn2.git
cd CAB432-assgn2
npm install
```
## Deployment
Run project (app.js) and use browser to view Projects.
```
npm start
```

Run test script to visualise status of API
```
npm test
```
